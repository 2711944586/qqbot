# 本地贴纸目录

把 GIF / PNG / JPG / WebP 文件放在这里，文件名（不含扩展名）就是贴纸名。

## 用法

群里发：
- `/sticker <文件名>` - 发指定贴纸
- `/sticker` (无参数) - 列出所有可用贴纸
- `/stickers` - 看所有可用 QQ 表情和本地贴纸

AI 对话中：
- bot 在合适的语境会自动用 `[名字]` 标签触发本地贴纸或 QQ 经典表情

## 文件命名建议

- 用容易记的中文/英文名，避免空格和特殊字符
- 例: `白给.gif`、`6657.png`、`太c了.gif`、`zywoo.png`

## 大小限制

- 单文件 ≤ 5MB（NapCat 限制）
- 推荐 GIF 不超过 2MB 否则发送慢
